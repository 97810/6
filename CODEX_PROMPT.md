# 给 Codex 的修改提示词

你现在接管一个 Windows C++/Win32 小工具项目，项目名为 `F2_NetworkSwitch_Rebuild`。这是根据一个 UPX 加壳 exe 的静态反编译结果整理出的可编译学习版，不是原作者源码。请基于现有代码修改，不要重写成完全无关的工程。

## 当前项目逻辑

- `src/main.cpp`：程序入口。
- `src/F2ClickerApp.h` / `src/F2ClickerApp.cpp`：核心逻辑。
- 程序创建一个 Win32 GUI 窗口，注册全局热键 `F2`。
- 第一次按 `F2`：获取鼠标所在窗口 `HWND` 和窗口内坐标，并显示“坐标/句柄/信息”。
- 后续按 `F2`：把绑定的窗口内坐标转换回屏幕坐标，然后通过 `SendInput` 模拟一次左键点击。
- “重置”按钮用于清空绑定，“退出”按钮用于关闭程序。
- `resources/app.manifest` 默认 `requireAdministrator`，如果仅调试可改成 `asInvoker`。

## 修改要求

请按照我的具体要求修改代码，并保持以下原则：

1. 保持项目可以用 VS Code + CMake 编译。
2. 不要直接运行未知 exe，不要依赖原始 exe。
3. 修改后必须说明改动了哪些文件、改动目的、如何编译运行。
4. 尽量保持代码简洁，中文 UI 不要乱码。
5. 如果新增功能，优先放在 `F2ClickerApp` 类中，避免把逻辑堆到 `main.cpp`。
6. 如果涉及网络开关功能，优先提供清晰、可审查的 Windows API 实现，不要使用隐藏命令或不可读的脚本。
